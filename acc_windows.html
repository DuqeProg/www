Accessibility of Windows Based on

Windows

  • 바탕화면
    • 바탕화면: Win M
    • 작업표시줄: Win T
    • 알림표시줄: Win B
    • 포커스해제: Ctrl Space
    • 스크린샷: Win Shift S
    • 선택된 아이템의 속성: Alt Enter
    • 팝업메뉴: Shift F10
  • OS일반
    • 시작: Ctrl Esc
    • 작업관리자: Ctrl Shift Esc
    • 취소: Ctrl Z
    • 재실행: Ctrl Y
    • 열린 응용프로그램 간 전환: Alt Tab
    • 모든 응용프로그램 내리기: Win D
    • 응용프로그램 종료: Alt F4
    • 탭 종료: Ctrl F4
    • 컴퓨터 잠금: Win L
    • 윈도우 로그인시 패스워드보임: Alt F8
  • Function
    • F3: 검색활성화
    • F4: 주소표시줄 포커스 추천사이트 드롭다운
    • F5: 새로고침 Ctrl R
    • F6: 주소표시줄 등 윈도우 구성요소 포커스(순환)
    • F7: 오피스 응용프로그램 문법검사 | 히라가나를 가타가나로 변환
    • F10: 풀다운 메뉴
  • 텍스트 이동
    • 줄의 시작: Home
    • 줄의 끝: End
    • 문서의 시작: Ctrl Home
    • 문서의 끝: Ctrl End
    • 다음단어로: Ctrl →
    • 이전단어로: Ctrl ←
    • 다음단락으로: Ctrl ↓
    • 이전단락으로: Ctrl ↑
    • 복사: Ctrl Insert
    • 붙여넣기: Shift Insert (값만 Ctrl Shift V)
  • 윈도우창
    • 뒤로: Alt ←
    • 앞으로: Alt →
    • 화면을 올림: Alt PgUp
    • 화면을 내림: Alt PgDn
    • 윈도우창의 포커스 없는 팝업메뉴: Alt Space
    • 주소표시줄 검색창 활성화: Ctrl E
    • Display the shortcut menu for the selected item.: Shift + F10

Sense Reader

프로그램 운용

  • 시작: Ctrl Alt S
  • 종료: Ins F4 | Ctrl Alt Shift F4
  • 일시정지: Alt Shift \
  • 메뉴창: Ctrl \ | Ctrl Alt S
  • 환경설정
    • 일반 - 환경열기: Ctrl Alt Shift O
    • 일반 - 기본환경열기: Ctrl Alt Shift B
    • 일반 - 환경저장: Ctrl Alt Shift S
    • 일반 - 다른이름으로환경저장: Ctrl Alt Shift A

정보표시

  • 타이틀바: Ctrl Shift T
  • 상태바: Ctrl Shift S
  • 윈도우창: Ctrl Shift W
  • 포커스위치: Ctrl Shift F

웹브라우징을 위한 탭키 설정

  • 탭키선택창: Ctrl Shift F9
  • 탭키선택토글: F9
  • 탭키on/off: Ctrl Shift F11 / F12
