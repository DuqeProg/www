Accessibility of Windows Based on

Windows

OS일반

  • 바탕화면: Win M
  • 작업표시줄: Win T
  • 알림표시줄: Win B
  • 포커스해제: Ctrl Space
  • 스크린샷: Win Shift S
  • 선택된 아이템의 속성: Alt Enter
  • 팝업메뉴: Shift F10
  • Delete the selected item without moving it to the Recycle Bin first.: Shift + Delete
  • 시작: Ctrl Esc
  • 작업관리자: Ctrl Shift Esc
  • 취소: Ctrl Z
  • 재실행: Ctrl Y
  • 열린 응용프로그램 간 전환: Alt Tab
  • 작업전환창: Ctrl Alt Tab
  • 모든 응용프로그램 내리기: Win D
  • 응용프로그램 종료: Alt F4
  • 탭 종료: Ctrl F4
  • 컴퓨터 잠금: Win L
  • 윈도우 로그인시 패스워드보임: Alt F8
  • 뒤로|앞으로: Alt ←|→
  • 화면을 올림|내림: Alt PgUp|PgDn
  • 윈도우창의 포커스 없는 팝업메뉴: Alt Space
  • 주소표시줄 검색창 활성화: Ctrl E
  • 팝업메뉴: Shift F10

Function

  • F2: 이름바꾸기
  • F3: 검색활성화
  • F4: 주소표시줄 포커스 추천사이트 드롭다운
  • F5: 새로고침 Ctrl R
  • F6: 주소표시줄 등 윈도우 구성요소 포커스(순환)
  • F7: 오피스 응용프로그램 문법검사 | 히라가나를 가타가나로 변환
  • F10: 풀다운 메뉴

텍스트 관리

  • 줄의 시작|끝: Home|End
  • 문서의 시작|끝: Ctrl Home|End
  • 다음|이전 단어로: Ctrl →|←
  • 다음|이전 단락으로: Ctrl ↓|↑
  • 모두선택: Ctrl A
  • 삭제: Ctrl D
  • 복사: Ctrl Insert
  • 붙여넣기: Shift Insert (값만 Ctrl Shift V)
  • 중복선택: Shift with any ↑↓←→

기타 등등

  • Set focus to the first icon in the Suggested actions menu.: Alt + A
  • Cycle through items in the order in which they were opened.: Alt + Esc
  • Perform the command for that letter.: Alt + underlined letter
  • When a group or tile is in focus on the Start menu, move it in the direction specified.: Alt + Shift + ↑↓←→
  • When a tile is in focus on the Start menu, move it into another tile to create a folder.: Ctrl + Shift + ↑↓←→
  • Resize the Start menu when it's open.: Ctrl + ↑↓←→
  • Select multiple individual items in a window or on the desktop.: Ctrl + ↑↓←→ (to move to an item) + Space
  • Select a block of text.: Ctrl + Shift with ↑↓←→
  • Switch the keyboard layout when multiple keyboard layouts are available.: Ctrl + Shift

Sense Reader

프로그램 운용

  • 시작: Ctrl Alt S
  • 종료: Ins F4 | Ctrl Alt Shift F4
  • 일시정지: Alt Shift \
  • 메뉴창: Ctrl \ | Ctrl Alt S
  • 환경설정
    • 일반 - 환경열기: Ctrl Alt Shift O
    • 일반 - 기본환경열기: Ctrl Alt Shift B
    • 일반 - 환경저장: Ctrl Alt Shift S
    • 일반 - 다른이름으로환경저장: Ctrl Alt Shift A

정보표시

  • 타이틀바: Ctrl Shift T
  • 상태바: Ctrl Shift S
  • 윈도우창: Ctrl Shift W
  • 포커스위치: Ctrl Shift F

웹브라우징을 위한 탭키 설정

  • 탭키선택창: Ctrl Shift F9
  • 탭키선택토글: F9
  • 탭키on/off: Ctrl Shift F11 / F12
